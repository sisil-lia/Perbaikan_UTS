## Deskripsi Aplikasi

Aplikasi Task Manager digunakan untuk mencatat dan mengelola tugas yang harus diselesaikan oleh pengguna.

## Fitur

* Melihat daftar tugas
* Menambahkan tugas baru
* Mengedit tugas yang belum selesai
* Menghapus tugas yang belum selesai
* Menandai tugas sebagai selesai
* Melihat tugas selesai berdasarkan rentang tanggal
* Dashboard statistik tugas
* Laporan jumlah tugas berdasarkan tanggal
* Testing menggunakan Laravel

## Teknologi

* Laravel 13
* SQLite
* Bootstrap 5
* Git
* GitHub

## Cara Menjalankan Aplikasi

Install dependency:

```bash
composer install
npm install
```

Migrasi database:

```bash
php artisan migrate
```

Menjalankan aplikasi:

```bash
php artisan serve
```

Akses melalui browser:

```text
http://127.0.0.1:8000
```
